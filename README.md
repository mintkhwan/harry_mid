## Usage
clone repository: `$ git clone https://github.com/mintkhwan/harry_mid.git`
### To install packages
```
$ sudo npm install
```
### To run Node.js server:
```
$sudo npm start
```
### To run the nightmarejs, mocha and chai test:
```
$sudo npm test
```
